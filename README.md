# АТОН — строительная компания

Сайт-одностраничник строительной компании **АТОН** (Челябинская область, Златоуст).

> Строим жизни, а не стены.

## О проекте

Лендинг основателя компании — Павла Зиберс. Одноэтажные дома под ключ с мебелью собственного производства.

## Стек

- Чистый HTML + CSS (без фреймворков)
- Шрифт: Inter (Google Fonts)
- Палитра: тёмно-зелёный + латунный акцент
- Адаптивная вёрстка

## Структура

- `index.html` — главная страница
- `style.css` — стили

## Запуск локально

```bash
git clone https://github.com/PavelZibers/aton-company.git
cd aton-company
open index.html
```
